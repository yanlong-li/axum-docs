Axum 是一个基于 Rust 语言的 web 应用框架，它专注于提供类型安全和异步处理能力。Axum 建立在 Hyper 和 Tower
之上，使得它能够轻松地处理并发请求。

## 异步运行时
专为 Tokio 构建，因此它天生就是与 Tokio 异步运行时兼容的。并且 Axum 使用标准的 Rust
异步特性，理论上它可以与任何提供了必要异步特性和 API 的异步运行时一起工作。这包括像 async-std 或 smol
这样的运行时，尽管可能需要一些额外的配置或兼容层。

在选择异步运行时时，重要的是要确保所选的运行时与你的应用需求以及 Axum 和其他依赖项的兼容性相匹配。Tokio 是最常见的选择，因为它直接支持
Axum，并且有一个广泛的生态系统和社区支持。如果你有特定的理由选择其他运行时，建议查看相关文档和社区资源以确保兼容性和性能。
