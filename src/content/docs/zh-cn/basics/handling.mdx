---
title: 处理器/控制器（Handling）
description: 处理器（Handling）又称控制器（Controller）
---

路由需要与处理器进行相关联，才能正确的处理请求。


## 参考案例
处理器可以是具名函数，也可以是匿名函数。生产环境更推荐具名函数，并合理规划目录结构。

无论采用何种方式，函数一定是需要异步的（`async`）。

### 匿名函数
```rust
Router::new().route("/hello", get(|| async { "Hello World!" }));
```

### 具名函数
```rust
Router::new().route("/hello", get(hello));

async fn hello() -> &'static str {
    "Hello World!"
}
```
