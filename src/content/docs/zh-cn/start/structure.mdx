<h2>目录和文件</h2>
<p>Cargo 创建的`二进制应用`提供一个默认的`应用模板`，结构如下：</p>
<ul>
    <li>`src/*` - 你的项目源代码</li>
    <li>`src/main.rs` - 二进制项目的入口文件</li>
    <li>`Cargo.toml` - Cargo项目配置文件</li>
</ul>


<h2>[热知识] WorkSpace</h2>
<p>另一种推荐的组织目录结构的方式是使用 workspace ，可以将多个 crates 分开管理。</p>
<p>为方便教程演示，这里不采用这种方式。详情可参考 <a href="https://github.com/yanlong-li/axum-web-tutorial" target="_blank">axum-web-tutorial</a></p>
